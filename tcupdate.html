AutoTorrentInspection is made for checking issues in such as file naming, text file encoding.

RPChecker is designed to compare the difference between two video clips by calculating the PSNR / SSIM value.

PS₀: For ffmpeg implementation, you can get the program from HERE.
PS₁: For VapourSynth implementation, you need to make sure that Python, VapourSynth, L-SMASH-Works, vs-ComparePlane are properly installed before using the RPChecker.
